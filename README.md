# FYP-Lab1-AI-module: student Marks Predicator

##Description
This is a bare-bones CLI based Linear Regression model that prects student's final based on Attendance, Assignment Marks, and quiz Marks.

##how to run
1) Install Requirements:'pip install pandas scikit-learn'
2) Run: 'py markspredicator.py'
3) Enter iputs when prompted


##Model
-Algorithm: Linear Regression
-Feauters: Attendance %, Assignment MArks, Quiz Marks
-Target: Final Marks out of 100
